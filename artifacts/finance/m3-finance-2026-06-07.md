# M3 finance flows smoke — 2026-06-07

Milestone: `M3 · Finance flows (the Cleo face)`

## Implemented

- Spend action opens a local finance breakdown panel over IndexedDB transactions.
- Spend action appends the local deterministic sass summary, then calls `/api/chat`
  with `financeContext` for QVAC narration.
- Stash action opens a local savings-goal flow backed by the encrypted finance
  store (`leclerc-finance`, store `goals`).
- Request action opens a Lightning invoice handoff panel and routes to
  `/es/billetera?invoice=...`.
- Wallet page reads the `invoice` query parameter and pre-fills the Lightning
  payment form after the demo wallet seed is generated in memory.

## Browser proof
